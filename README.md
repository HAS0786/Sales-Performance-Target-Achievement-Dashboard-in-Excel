# Sales Performance & Target Achievement Dashboard

## Project Overview

This project is an interactive Excel dashboard designed to analyze sales executive performance across multiple cities. The dashboard helps track sales achievements, compare employee performance, and identify top and bottom performers based on sales and target completion rates.

The dashboard provides a simple and user-friendly way to monitor business performance and support data-driven decision-making.

---

## Objectives

* Analyze sales executive performance
* Track target achievement percentages
* Identify top-performing employees
* Highlight underperforming sales executives
* Compare performance across different cities
* Create an interactive business reporting dashboard
